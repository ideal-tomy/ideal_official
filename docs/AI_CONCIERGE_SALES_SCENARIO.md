# AIコンシェルジュ — 営業デモ用シナリオ（サイト案内）

最終更新: 2026-07-23  
関連: [`AI_CONCIERGE_REDESIGN.md`](./AI_CONCIERGE_REDESIGN.md)

## 目的

> 聞きたいことを自然文で言うと、正しいページに連れていく

を 5 分で見せる。

---

## 推奨ルート

```
TOP
 → デモギャラリーを軽く触る（任意）
 → 右下ロボット（案内）を開く
 → 「金額の目安が知りたい」またはチップ「金額・見積もり」
 → 概算見積もりへ のリンクを見せる
 → 「他社と何が違う？」→ Reason（/#reason）
 → （任意）ページ末尾までスクロール → Footer ロボットのあいさつ
```

---

## トーク（短版）

1. **入口**  
   「普通のサイトはメニューを探すか、フォームに書くだけです。ideal は案内ロボットに聞くと、サイト内の行き先を返します。」

2. **テンプレ案内**  
   チップまたは「デモを触りたい」→ Gallery リンク。  
   「よく聞かれることはテンプレで即案内。だからコストも抑えつつ、サイト外の話には答えません。」

3. **金額**  
   「金額はチャットで作らず、見積ページへ送ります。」

4. **違い**  
   「他社との違いは Reason にまとめてあるので、端的に答えてそこへ案内します。」

5. **Footer**  
   「最後まで見た人には、同じキャラが一言声をかけます。押しはデモ／概算／相談の直リンクです。」
